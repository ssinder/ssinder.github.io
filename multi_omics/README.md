# 油菜多组学数据库 (OilseedDB)

油菜多组学数据查询、可视化和分析平台前端页面。

## 项目简介

油菜多组学数据库是一个现代化的前端应用，整合了高质量的油菜全基因组、转录组、表观遗传组、代谢组等多组学数据，提供数据查询、可视化分析和在线分析工具。

## 主要功能

- **数据查询** - 多维度查询数据集，支持条件筛选、模板保存与复用
- **可视化分析** - UMAP降维、空间热图、联合图等交互式可视化
- **在线工具** - 差异表达分析、富集分析、细胞通讯分析、轨迹分析
- **数据下载** - 灵活的数据下载服务，支持多种格式（H5等）
- **全局搜索** - 快速检索基因、样本、DOI等信息

## 技术栈

- **前端技术**：原生 HTML5、CSS3、JavaScript（ES6+）
- **模块化架构**：模块化 JavaScript 组织
- **主题系统**：5种预设主题（清新薄荷绿、经典深蓝夜、学术浅灰、活力青绿、高对比度）
- **API 通信**：RESTful API 集成

## 功能模块

### 1. 数据查询 (QueryModule)
- 多条件组合查询
- 查询模板保存与加载
- 分页和排序
- 网格/列表视图切换

### 2. 可视化分析 (VisualizationModule)
- UMAP 降维可视化
- 空间热图展示
- 联合图分析
- 可配置参数（点大小、透明度、配色方案）
- 图片导出功能

### 3. 在线工具 (ToolsModule)
- 差异表达分析
- GO/KEGG 富集分析
- 细胞通讯分析
- 轨迹分析

### 4. 数据下载 (DownloadModule)
- 数据类型筛选
- 多种格式支持
- 下载历史记录
- 批量下载

### 5. 用户管理 (UserModule)
- 用户注册登录
- 个人中心
- 收藏管理

## API 接口

后端 API 基础地址：`http://127.0.0.1:5001`

主要接口包括：
- `/api/query/*` - 数据查询相关
- `/api/visualization/*` - 可视化分析相关
- `/api/tools/*` - 在线工具相关
- `/api/download/*` - 数据下载相关
- `/api/user/*` - 用户管理相关
- `/api/search/*` - 全局搜索

## 主题系统

系统支持 5 种视觉主题，可在界面右上角切换：
1. 清新薄荷绿（默认）
2. 经典深蓝夜
3. 清新学术浅灰
4. 活力科研青绿
5. 高对比度无障碍

## 使用方式

1. 直接在浏览器中打开 `index.html`
2. 启动后端服务（确保 API 可用）
3. 使用顶部导航访问不同功能模块

## 开发说明

- 项目采用纯前端实现，无需构建工具
