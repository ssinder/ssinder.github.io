# 大脑单细胞与空间转录组数据库

大脑单细胞和空间转录组数据管理与可视化分析的前端平台。

## 项目简介

该项目旨在为神经科学研究人员提供一个直观、高效的数据查询、可视化和分析工具，支持单细胞转录组、空间转录组及多组学整合数据的全面分析。

## 主要功能

### 1. 数据查询
- 多维度筛选：数据类型、测序平台、物种、脑区、发育阶段
- 基因和表达量阈值查询
- 查询条件保存与复用

### 2. 可视化分析
- UMAP降维可视化
- 空间转录组热图展示
- 联合关联图分析
- 可调参数：点大小、透明度、配色方案
- 支持导出高质量图片

### 3. 在线分析工具
- 差异基因分析（DEG）
- 富集分析（GO/KEGG/Reactome）
- 细胞通讯分析
- 轨迹分析

### 4. 数据下载
- 支持多种格式：H5、H5AD、CSV、Excel
- 批量下载功能

## 技术架构

- **前端框架**：原生HTML5 + CSS3 + JavaScript
- **主题系统**：支持4种专业配色方案
  - 经典科研蓝调
  - 清新学术浅灰
  - 活力科研青绿
  - 高对比度无障碍
- **响应式设计**：适配多种屏幕尺寸

## 数据维度

| 维度 | 支持选项 |
|------|---------|
| 数据类型 | 单细胞转录组、空间转录组、多组学整合 |
| 测序平台 | 10x Genomics、Smart-seq2、Visium、Slide-seq |
| 物种 | 人类、小鼠、大鼠 |
| 脑区 | 海马体、大脑皮层、纹状体、小脑 |
| 发育阶段 | 胚胎期、出生后、成年、老年 |

## 使用说明

1. 直接在浏览器中打开 `index.html` 即可访问
2. 通过顶部导航栏切换不同功能模块
3. 使用左侧面板进行参数配置和筛选
4. 支持多主题切换以适应不同使用场景
