# 大脑单细胞与空间转录组数据库网页设计方案（功能-布局精准对应版）
**核心原则**：基于顶刊级生物信息学数据库（CancerSEA、UCSC Cells、Stomics等）经典布局逻辑，构建“**顶部全局导航-左侧功能操控-右侧内容展示-底部合规信息**”的四层架构，实现**每个功能模块与页面布局区域一一对应**，确保科研人员操作路径清晰、功能调用直观。

## 一、整体布局框架（核心映射基础）
采用“**固定导航+可折叠功能区+多标签内容区+固定底部**”的经典科研数据库布局，所有功能均锚定至具体布局区域，无游离功能点。

| 布局层级 | 区域名称 | 占比/位置 | 核心功能定位 | 对应参考网页 |
|----------|----------|-----------|--------------|--------------|
| 第一层 | 顶部导航区 | 100%通栏（固定悬浮） | 全局功能入口、身份管理、快速检索 | CancerSEA、UCSC Cells |
| 第二层 | 左侧功能区 | 22%-25%宽度（可折叠） | 功能参数配置、条件输入、工具选择 | CancerSEA、Cancerscem |
| 第三层 | 右侧内容区 | 75%-78%宽度（多标签页） | 功能结果展示、数据可视化、详情查看 | Stomics、SpatialDB |
| 第四层 | 底部信息区 | 100%通栏（固定） | 数据合规、引用、技术支持 | CNCB系列数据库 |

## 二、功能需求与页面布局精准对应（核心部分）
### 2.1 顶部导航区（全局功能层）
**布局结构**：左侧数据库LOGO+名称 → 中间核心导航菜单 → 右侧全局搜索框+个人中心入口
**对应功能模块**：

| 布局子区域 | 具体功能 | 功能描述 | 操作路径 |
|------------|----------|----------|----------|
| LOGO+名称 | 首页跳转 | 点击LOGO快速返回数据库首页，展示数据库核心定位（大脑单细胞+空间转录组整合平台） | 顶部左侧 → 点击LOGO → 跳转首页 |
| 核心导航菜单 | 功能模块切换 | 包含“首页、数据查询、可视化分析、在线工具、数据下载、帮助中心”6个核心入口，点击切换至对应功能页面 | 顶部中间 → 点击菜单选项 → 右侧内容区加载对应页面 |
| 全局搜索框 | 跨维度快速检索 | 支持基因名/基因ID/样本编号/DOI/脑区名称的全局搜索，输入时自动联想补全，点击结果直接跳转至详情页 | 顶部右侧 → 输入关键词 → 点击搜索/回车 → 右侧内容区显示搜索结果 |
| 个人中心入口 | 身份管理与数据收藏 | 支持登录/注册、查看收藏的数据集/查询模板/可视化结果、管理数据上传记录 | 顶部最右侧 → 点击头像/“个人中心” → 弹出下拉菜单/跳转个人中心页 |

### 2.2 左侧功能区（功能操控层）
**布局结构**：可折叠侧边栏，按“当前功能页面”动态切换子面板，面板内按“功能分组”折叠/展开，底部固定功能按钮
**对应功能模块**：左侧功能区与右侧内容区的“标签页”联动，不同标签页对应不同的左侧子面板，核心对应关系如下：

#### 2.2.1 数据查询页 → 左侧“多维度查询条件面板”
**布局子面板**：数据维度子面板、样本维度子面板、分析维度子面板、关联维度子面板、条件管理子面板（底部固定）
**功能-布局对应**：

| 左侧子面板 | 布局位置 | 对应功能 | 功能操作 |
|------------|----------|----------|----------|
| 数据维度子面板 | 左侧顶部（默认展开） | 数据类型/测序平台/数据属性/数据版本筛选 | 勾选/下拉选择 → 自动加入查询条件池 |
| 样本维度子面板 | 数据维度下方（可折叠） | 物种/脑区/发育阶段/样本状态/细胞类型筛选 | 脑区支持“父级（如海马体）→子级（CA1/CA3）”层级选择，细胞类型支持多选 |
| 分析维度子面板 | 样本维度下方（可折叠） | 基因输入/表达量阈值/聚类参数筛选 | 基因输入框支持批量粘贴，表达量阈值支持手动输入范围 |
| 关联维度子面板 | 分析维度下方（可折叠） | 文献DOI/期刊/研究团队筛选 | 输入DOI自动匹配关联数据集 |
| 条件管理子面板 | 左侧底部（固定） | 条件保存/加载/重置/查询 | 点击“保存模板”命名存储查询条件，点击“查询”触发右侧结果展示 |

| 数据筛选子面板 | 图表类型下方 | 数据集选择/基因选择/细胞类型/空间区域筛选 | 选择已查询的数据集，输入目标基因，勾选需展示的细胞类型/空间区域 |
| 参数调整子面板 | 数据筛选下方 | 图表样式/表达阈值/配色/坐标轴设置 | 调整点大小、透明度、表达量显示范围 → 右侧图表实时刷新 |
| 导出设置子面板 | 左侧底部（固定） | 图表格式/分辨率/数据导出选择 | 选择PNG/SVG/PDF格式，设置4K分辨率 → 点击“导出”触发下载 |

#### 2.2.3 在线工具页 → 左侧“分析工具参数面板”
**布局子面板**：工具选择子面板、数据导入子面板、算法参数子面板、运行/导出子面板（底部固定）
**功能-布局对应**：

| 左侧子面板 | 布局位置 | 对应功能 | 功能操作 |
|------------|----------|----------|----------|
| 工具选择子面板 | 左侧顶部 | 差异基因分析/富集分析/细胞通讯分析选择 | 选择工具 → 下方面板自动切换对应参数项 |

### 2.3 右侧内容区（核心展示层）
**布局结构**：多标签页设计（支持新建/关闭/切换），每个标签页对应一个功能结果，标签页内按“概览-详情-操作”三层结构展示
**对应功能模块**：右侧内容区的标签页与左侧功能区联动，每个标签页承载一个完整的功能结果流，核心对应关系如下：

#### 2.3.1 “数据查询结果”标签页 → 对应“多维度查询功能”
**标签页内布局**：顶部概览面板 → 中间数据展示区（卡片/表格切换） → 底部分页与二次筛选栏
**功能-布局对应**：

| 标签页子区域 | 布局位置 | 对应功能 | 功能呈现 |
|--------------|----------|----------|----------|
| 概览面板 | 标签页顶部（通栏） | 查询结果统计 | 显示匹配数据集总数、物种/脑区/数据类型分布饼图、热门数据集Top5 |
| 数据展示区 | 标签页中间（核心） | 数据集卡片/详细表格展示 | 网格布局显示数据卡片（含核心信息+快捷按钮），列表布局显示详细表格（支持排序/隐藏列） |
| 二次筛选栏 | 标签页底部（通栏） | 结果二次筛选/排序 | 按“发表年份/引用次数/数据完整性”排序，按“物种/数据类型”二次筛选 |
| 快捷操作按钮 | 每个卡片/表格行 | 可视化/下载/详情跳转 | 点击“可视化”跳转至可视化分析页，点击“下载”触发单数据集下载，点击“详情”跳转至数据集详情页 |

## 三、实现思路（功能-布局联动逻辑）
### 3.1 布局与功能的联动机制
1.  **页面切换联动**：点击顶部导航菜单时，左侧功能区自动加载对应页面的子面板，右侧内容区新建标签页并加载对应功能的基础框架（如点击“数据查询”，左侧加载查询条件面板，右侧加载查询结果标签页）。
2.  **参数调整联动**：左侧功能区的参数输入/选择操作，实时传递至右侧内容区，无需刷新页面即可更新结果（如在左侧调整基因表达阈值，右侧可视化图表实时重绘）。
3.  **标签页联动**：右侧内容区的标签页支持“跨功能跳转”（如从数据查询结果标签页点击“可视化”，直接新建可视化分析标签页，并自动带入当前数据集参数）。

### 3.2 核心功能落地逻辑（通俗版）
1.  **查询功能落地**：用户在左侧查询条件面板设置筛选条件 → 点击“查询”按钮 → 系统筛选数据库并将结果发送至右侧查询结果标签页 → 右侧按“概览-卡片-表格”层级展示，支持二次操作。
2.  **可视化功能落地**：用户从查询结果中选择数据集 → 点击“可视化” → 左侧切换为可视化参数面板，右侧新建可视化标签页 → 用户调整参数，右侧图表实时更新 → 点击“导出”保存图表/数据。
3.  **详情功能落地**：用户点击数据集“详情” → 右侧新建详情标签页，左侧切换为详情操作面板 → 右侧分标签展示数据集全维度信息，支持文件筛选、批量下载、关联文献跳转。
