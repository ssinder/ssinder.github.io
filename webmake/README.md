# 大脑单细胞与空间转录组数据库 - 项目进展记录

## 项目概述
本项目基于需求文档REQUIREMENT.MD，开发了一个顶刊级生物信息学数据库的前端界面，实现了大脑单细胞与空间转录组数据的查询、可视化、分析和下载功能。

## 项目目标
构建一个"顶部全局导航-左侧功能操控-右侧内容展示-底部合规信息"的四层架构，实现每个功能模块与页面布局区域一一对应，确保科研人员操作路径清晰、功能调用直观。

- 高亮色：#6BAED6（亮蓝）- 数据高亮、hover提示

**适用场景**：主打数据权威性、文献级引用的数据库，适合传统脑科学研究人员
