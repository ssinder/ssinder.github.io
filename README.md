# Bio-Omics Web Collection

以下是我做项目过程中搭建的一些网站的前端部分，多为ai辅助编程，欢迎大家提出建议。

## 项目列表

### 1. [油菜多组学数据库 (OilseedDB)](https://ssinder.github.io/multi_omics/)
整合油菜全基因组、转录组、表观遗传组、代谢组等多组学数据，提供数据查询、可视化分析和在线工具。

**访问地址**: https://ssinder.github.io/multi_omics/

### 2. [PDB 蛋白质结构搜索平台](https://ssinder.github.io/pdb_search/)
蛋白质结构数据搜索与分析平台，支持关键词搜索、布尔查询、序列过滤和数据导出。

**访问地址**: https://ssinder.github.io/pdb_search/

### 3. [大脑单细胞与空间转录组数据库](https://ssinder.github.io/scrna_show/)
神经科学研究数据管理平台，支持单细胞转录组、空间转录组及多组学整合数据的查询与可视化。

**访问地址**: https://ssinder.github.io/scrna_show/

### 4. [分泌蛋白互作信息查询系统](https://ssinder.github.io/target_search/)
分泌蛋白互作网络查询与可视化工具，基于 D3.js 展示交互式蛋白互作关系图。

**访问地址**: https://ssinder.github.io/target_search/

## 技术特点

- 纯前端实现，无需后端服务器
- 响应式设计，适配多种设备
- 多个配色主题可选
- 支持数据导出功能
- 交互式可视化图表

## 部署说明

本项目可直接部署至 GitHub Pages。每个子项目为独立模块，可单独访问。
