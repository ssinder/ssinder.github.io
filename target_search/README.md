# 分泌蛋白互作信息查询系统

Secreted Protein Interaction Information Query System

## 项目简介

这是一个用于查询分泌蛋白（Secreted Protein）互作信息的可视化查询系统。用户可以通过基因 ID 查询蛋白的详细信息及其与其他蛋白的互作关系，并以网络图的形式直观展示蛋白互作网络。

## 主要功能

- **蛋白查询**：通过基因 ID（Gene ID）搜索蛋白信息，支持模糊匹配和自动补全
- **详细信息展示**：显示蛋白的 UniProt ID、蛋白名称、长度、催化活性、功能、定位等信息
- **互作网络可视化**：使用 D3.js 绘制交互式蛋白互作网络图
  - 节点大小和颜色反映互作置信度
  - 支持缩放、平移、拖拽等交互操作
  - 可导出网络图为图片
- **性能优化**：使用 IndexedDB 本地缓存和内存索引，实现快速查询

## 技术栈

- **前端框架**：原生 JavaScript（模块化设计）
- **可视化**：D3.js v7
- **数据存储**：IndexedDB（浏览器本地缓存）
- **样式**：CSS3

## 模块说明

- **app.js**：应用入口，处理用户输入、查询逻辑和模块协调
- **data.js**：负责加载数据、构建索引、提供查询接口（含模糊搜索）
- **ui.js**：处理 DOM 操作，渲染蛋白卡片和互作列表
- **network.js**：基于 D3.js 实现蛋白互作网络的可视化和交互

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 在搜索框中输入基因 ID（如 `SS1G_02266`）
3. 系统会自动显示匹配的建议
4. 点击查询后，展示蛋白详情及互作网络图
5. 可通过鼠标滚轮缩放网络图，点击节点查看详细信息

## 数据说明

- 蛋白信息包含：UniProt ID、蛋白名称、序列长度、催化活性、功能描述、细胞定位、基因本体论（GO）注释等
- 互作信息包含：互作蛋白、置信度分数等信息
- 置信度等级：高（≥700）、中（400-699）、低（<400）

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）的最新版本。
